Poc 3 - Media Queries

Media Queries

Media Queries são uma funcionalidade do CSS usada para aplicar diferentes estilos a um documento com base em características do dispositivo ou do ambiente em que a página está sendo visualizada. Elas são essenciais no design responsivo, permitindo que layouts se adaptem automaticamente a diferentes tamanhos de tela, resoluções, orientações (como landscape e portrait), e até mesmo ao contexto de impressão.

Com media queries, é possível especificar condições, como:

  • Largura de tela: Adaptar o layout para dispositivos pequenos (smartphones), médios (tablets) ou grandes (desktops).
  • Orientação: Ajustar o design dependendo se o dispositivo está em modo retrato (portrait) ou paisagem (landscape).
  • Impressão: Criar estilos específicos para quando o usuário desejar imprimir o conteúdo.

Elas permitem um desenvolvimento web mais flexível e acessível, proporcionando uma melhor experiência ao usuário em diferentes contextos e dispositivos.
